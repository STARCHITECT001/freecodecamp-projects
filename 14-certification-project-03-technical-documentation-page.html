HTML and CSS

HTML (HyperText Markup Language) and CSS (Cascading Style Sheets) are foundational technologies for creating webpages.

HTML defines the structure, while CSS handles the visual presentation.

<div>This is a div</div> p { color: red; }

Here’s how you can style a paragraph:

p { font-size: 16px; }

CSS can also be used for responsive design:

@media screen and (max-width: 600px) { /* styles for small screens */ }
  • HTML defines structure.
  • CSS defines styling.
  • HTML uses elements such as <div>, <p>, <header>.
  • CSS applies to HTML elements using selectors.
  • Responsive design ensures readability on all screen sizes.
JavaScript and Java

JavaScript is a programming language used to create interactive effects within web browsers.

Java is a general-purpose programming language used for building applications across various platforms.

const x = 10; function greet() { alert("Hello World!"); }

JavaScript is often used for front-end development, while Java is used for back-end services.

public class HelloWorld { public static void main(String[] args) { System.out.println("Hello World"); } }
  • JavaScript is for client-side scripting.
  • Java is used for server-side applications.
  • JavaScript can manipulate HTML and CSS.
  • Java is compiled, while JavaScript is interpreted.
  • Both are object-oriented languages.
Python Programming

Python is a versatile programming language used for a wide variety of applications, from web development to machine learning.

It is known for its readability and simple syntax.

print("Hello, World!") def add(a, b): return a + b

Python has a large community and a variety of libraries for almost every task.

import numpy as np
  • Python is easy to learn for beginners.
  • It supports multiple programming paradigms.
  • Python has extensive libraries for data science.
  • It’s an interpreted language.
  • Python is often used for automation tasks.
Database Management

Database management systems (DBMS) are used to store and retrieve data efficiently.

Popular DBMS include MySQL, PostgreSQL, and MongoDB.

SELECT * FROM users; CREATE TABLE students (id INT, name VARCHAR(100));

Databases can be relational (SQL) or non-relational (NoSQL).

db.collection.find()
  • SQL databases use structured query language.
  • NoSQL databases offer more flexible data models.
  • Indexes improve query performance.
  • Normalization is used to avoid redundancy in relational databases.
  • ACID properties ensure data integrity in transactions.
Web Development Tools

Web developers use various tools to streamline their workflow.

Some common tools include code editors, version control systems, and build tools.

git commit -m "Initial commit" npm install react

These tools help in managing code, dependencies, and tasks.

webpack --config webpack.config.js
  • Git is a version control system.
